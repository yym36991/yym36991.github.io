05、我的身份，类型格式

数据类型
整型 int() 将合法字符串数值或浮点数转成整数
字符串型 str() 将任意类型转成字符串型
浮点型 float() 将合法字符数值或整数转成浮点数
布尔型 bool() 将任意类型转成bool型

print(type(False))
注意大写，
条件表达式

b = “123”
print(b + 4)
b = int(“123”)
print(b + 4)
能转的才能转。
字符串拼接

格式输出：
%d,%f,%s
name = “小明”
age = 3.123456
height = 1.23456789
print(“我的年龄是：” + name + “ 我的年龄是：” + str(age) +” 我的身高是：” + str(height))
print(“我的年龄是：%s, 我的年龄是：%d, 我的身高是：%10.2f” % (name, age, height))
